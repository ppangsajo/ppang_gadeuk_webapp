1. git clone으로 프로젝트를 다운로드 합니다.
2. 다운로드한 폴더로 이동합니다.
3. `npm install` 을 통해 프로젝트에 필요한 Node 라이브러리를 다운로드 합니다.
4. `npm run start` 명령어로 프로젝트를 실행합니다.
